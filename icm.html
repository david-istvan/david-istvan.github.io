I'm developing a modeling tool for managing inconsistencies in engineering processes. The tool is built on top of Eclipse. Installing it requires the following steps.

Installation

  1. Download the Eclipse Oxygen Modeling package.
  2. Install Sirius 5.0.1.
  3. Install VIATRA 1.6.0.
  4. Install the tool from HERE.

External dependencies

  1. Install Python3 and SymPy for for symbolic inconsistency checking.
  2. Install Matlab/Simulink (last tested version: 2016b) and set up your environment for the Java API as described here.
    (Typically, you'll only have to add matlabroot/bin/ARCH to your system environment variable, where ARCH is your computer architecture. For example, win64 for 64â€“bit Windows machines, maci64 on Macintosh, or glnxa64 on Linux.)
